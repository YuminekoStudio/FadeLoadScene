# Fade Load Scene

## 概要
シーンの切り替えを非同期で行ないます。  
フェードを入れることも可能です。
!["gif"](https://user-images.githubusercontent.com/15545226/62416174-aefe3b00-b670-11e9-82b6-91e25a77bb18.gif)

## 使い方

1. `Assets/Plugins` 以下を導入したいプロジェクトに入れる
2. `LoadScene.cs` をテキトーなGameObjectにアタッチ
3. `FadeLoad()` メソッドを呼び出す
